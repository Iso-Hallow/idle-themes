# Пользовательские темы для Python IDLE

## Введение

Данный репозиторий содержит коллекцию пользовательских цветовых схем (тем) для стандартной интегрированной среды разработки Python IDLE, а также для ее расширения IDLEX. Темы были адаптированы для обеспечения комфортной работы в условиях недостаточного освещения и включают исправления для корректного отображения всех элементов интерфейса, в частности, поля ввода данных функции `input()`.

## Представленные темы

В настоящее время коллекция включает две темы:

1.  **FLASH black:** Тема с высоким контрастом, основанная на цветовой палитре Monokai. Отличается яркими акцентами для ключевых слов и строк на глубоком черном фоне.
2.  **Dracula:** Популярная темная тема с более мягкой, фиолетово-синей палитрой. Обеспечивает комфортное восприятие без излишней нагрузки на зрение.

## Установка и настройка

Процесс установки заключается в добавлении кода тем в конфигурационный файл IDLE. Пожалуйста, следуйте приведенной ниже пошаговой инструкции.

### Шаг 1: Определение местоположения конфигурационного файла

Конфигурационный файл с настройками тем носит имя `config-highlight.cfg`. Для определения его точного местоположения необходимо выполнить следующие действия:

1.  Запустить среду IDLE.
2.  В главном меню выбрать `Options` → `Configure IDLE`.
3.  В открывшемся диалоговом окне перейти на вкладку `Highlights`.
4.  Путь к активному файлу `config-highlight.cfg` будет указан в информационном блоке данного окна.

Стандартное местоположение файла в операционной системе Windows:
`C:\Users\%USERNAME%\.idlerc\config-highlight.cfg`

### Шаг 2: Добавление кода тем

Необходимо интегрировать код новых тем в существующий конфигурационный файл.

1.  Откройте файл `config-highlight.cfg`, найденный на предыдущем шаге, в любом текстовом редакторе.
2.  Откройте соответствующий файл `.txt` из данного репозитория (`FLASH_black.txt` или `Dracula.txt`).
3.  Скопируйте все содержимое файла темы.
4.  Вставьте скопированный текст в **конец** вашего файла `config-highlight.cfg`.

**Примечание:** Не рекомендуется полностью заменять существующий файл `config-highlight.cfg` моим, так как это может привести к удалению ваших собственных, ранее настроенных тем. Добавление в конец файла является безопасной операцией.

### Шаг 3: Активация темы в IDLE

1.  Полностью закройте и снова запустите IDLE для применения изменений.
2.  Перейдите в меню `Options` → `Configure IDLE` → `Highlights`.
3.  В выпадающем списке под заголовком "Select a Built-in or Custom Theme" теперь должны появиться новые темы: "FLASH black" и "Dracula".
4.  Выберите желаемую тему из списка.
5.  Нажмите кнопку `Apply` для применения и `OK` для сохранения выбора.

Настройка завершена. Среда разработки теперь будет использовать выбранную вами цветовую схему.

## Происхождение и адаптация

Данные цветовые схемы основаны на работах, оригинальные источники которых в настоящее время могут быть недоступны.

*   **FLASH black:** Эта тема является восстановленной и адаптированной версией классической темной темы. В оригинальный код было внесено исправление (параметр `todo-background`) для обеспечения корректной работы с современными версиями Python IDLE.
*   **Dracula:** Адаптация популярной кросс-платформенной темы Dracula для среды IDLE.

Цель данного репозитория — не присвоить авторство, а **сохранить, адаптировать и сделать доступными** эти удобные цветовые схемы для сообщества, а также задокументировать решение распространенных проблем с их отображением.
